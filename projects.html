Infiniforge: Procedurally Generating 3D swords with JavaScript

Inspired by the then upcoming release of 'No Man's Sky' and my interest in blacksmithing, I started this project. My goal was to finish and submit it to the Reddit PCG challenge linked below. Since then, I have been continuously tinkering with this project in my free time. This project served as my introduction to both procedurcal content generation as well as computer graphics. The result is a NodeJs server that exports 3D meshes (ThreeJs) as glTF JSON via a REST API. These glTF models can be imported into Unity and other software that support the format.

Past Research

Auditory P300 Event-Related Potentials for Virtual Reality

This was my masters research project. The goal was to explore the use of an auditory P300 BCI paradigm for enabling users to select objects in virtual reality. What I ended with was an EEG-hardware agnostic Unity project that could be configured to run different P300 paradigms in virtual reality. The while runnning the unity scene EEG data is streamed from the headset and written out to files to be processed. For example, one is able to take recorded EEG data use a machine learning classifier to make predictions as to what object the user wants to select. Pictured is the virtual apartment environment with three objects used during testing.

Github Link: Audio P300 ERP

Teaching Agents to Box with Reinforcement Learning in Unity

I threw this project together over the course of a week as an entry in Unity's first challenge for their new 'Ml-Agents' machine learning workflow. I had tossed around the idea of using machine learning to train AI agents in fighting games, but I had never gotten around to it until then. The agents learn via Reinforcement Learning and are given a simple policy that positively rewards landed punches on the opponent, and negatively awards taking damage. Other behaviors such as successful blocking, and advancing on the opponent are rewarded as well.

Challenge Link: Unity Machine Learning Agent Challenge
